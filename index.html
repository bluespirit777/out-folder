          Building a Home Network with Cisco Packet Tracer

          In this project, I designed and configured a home network using Cisco Packet Tracer, simulating a residential setup with a Cisco 2911 router, a Cisco 2960 switch, and multiple end devices. I configured the router with a static IP address and assigned IPs to the end devices, ensuring seamless connectivity through extensive testing. Additionally, I implemented basic security measures by setting up Access Control Lists (ACLs) on the router.
